Dependencies:
ESP32 LoRa Libs from Heltec : use the guide from https://docs.heltec.org/en/node/esp32/esp32_general_docs/quick_start.html but use the link https://github.com/Heltec-Aaron-Lee/WiFi_Kit_series/releases/download/0.0.9/package_heltec_esp32_index.json on the packet manager to get the older version of the lib(v0.0.9)
TinyGpsPlus-ESP32 : download from arduino IDE (v0.0.2)
