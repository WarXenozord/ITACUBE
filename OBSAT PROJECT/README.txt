Software utilizado na OBSAT 2022.

Dependencias:
HttpClient by Adrian McEwen (Usada para o cliente que envia as requisicoes http do cubesat) - Download via arduino IDE
Pion Educational Kits by Pion Labs (Usada em praticamente todos os sistemas do cubesat) - https://github.com/pion-labs/pion-educational-kits/wiki/Software (Utilizar versao que consta neste git)
ArduinoJson by Benoit Blachon (Usada para facilitar a criacao de mensagems em JSON) - Download via arduino IDE
NeoGps by SlashDevin (Usada para traduzir os dados brutos do GPS) - Download via arduino IDE
Heltec by Aaron Lee (Usada para os radios LoRa) - https://heltec-automation-docs.readthedocs.io/en/latest/esp32/quick_start.html
OneWire by  Jim Studt, Tom Pollard - usada para o sensor de temperatura externo - Download via arduino IDE

Util:
Zenith Server (Usado para testar o sistema que enviava arquivos JSON para a wifi do balao) - https://github.com/zenitheesc/serverd
Gnss Toolkit (Usado para programar o gps que foi comprado mas era falso) - https://forum.arduino.cc/t/gps-module-change-update-rate-to-10hz/665212
